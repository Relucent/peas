介绍
------------------------------------

### framework
	
	整个项目的基础架构服务提供者，依赖于Spring容器的支持，添加Hibernate支持。

### common
	 工具类，资源及其他公用服务提供者，主要依赖于 apache common 的相关支持。


### apps 项目用例

	apps-server 服务提供端，用例apps所有的业务逻辑请求，都应该在这里进行处理，用例是进行远程调用实现。
